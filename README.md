# ChatGPT Calculator Integration

Video link: https://youtube.com/shorts/8WlpIg4gDe0

## Overview
This project demonstrates integrating a ChatGPT language model into a standard calculator interface. It combines hardware understanding, software programming, and AI interaction to create a compact, intelligent device. The goal was to explore how AI can operate within constrained hardware environments.

## Features
- **AI Interaction:** Users can ask questions or solve problems through the calculator interface.
- **Embedded Logic:** Efficient handling of inputs and responses to fit within calculator memory constraints.
- **Real-time Processing:** Immediate responses without external devices, demonstrating hardware-software integration.
